S1 + Turbowarp

S1 + Turbowarp.
Η ενεργοποίηση του πρόσθετου γίνεται σε 2 βήματα:
(0) Αντιγραφή του κώδικα s1-arduino.ino και φόρτωση του μέσω του Arduino IDE. Αυτό το βήμα γίνεται μια φορά, εφόσον δεν φορτωθεί κάποιος άλλος κώδικας στο Arduino.

Παρατηρήσh: Η εγκατάσταση των libraries DHT11 και Adafruit Neopixel, χρειάζεται μόνο μια φορά στο Arduino IDE
(1) Αντιγραφή του αρχείου s1-tubowarp.js. Επίσκεψη στη σελίδα του Turbowarp Editor, κλικ στο εικονίδιο "Προσθήκη επέκτασης", Custom Extension, Text και επικόλληση μέσα στο text box. Δεν ξεχνάμε να κάνουμε check στο Run without sandbox, πριν πατήσουμε το κουμπί Load.

            // First code snippet (inline fallback)
            function greet(name) {
            console.log("Hello, " + name);
            }
            greet("Alice");

Gigo + Turbowarp

Gigo + Turbowarp.
Η ενεργοποίηση του πρόσθετου γίνεται σε 2 βήματα:
(0) Αντιγραφή του κώδικα gigo-micropython.py και φόρτωση του μέσω του Micropython Microbit Web IDE. Διαγραφή του κώδικα που βρίσκεται μέσα στο Micropython Microbit Web IDE και επικόλληση του κώδικα του αρχείου gigo-micropython.py (κατά προτίμηση από browser της οικογένειας Google Chrome για απευθείας σύνδεση μέσω WebUSB που δυστυχώς δεν υποστηρίζεται στο Mozilla Firefox). Αυτό το βήμα γίνεται μια φορά, εφόσον δεν φορτωθεί κάποιος άλλος κώδικας στο Microbit μέσω upload.

(1) Αντιγραφή του αρχείου gigo-turbowarp.js. Επίσκεψη στη σελίδα του Turbowarp Editor, κλικ στο εικονίδιο "Προσθήκη επέκτασης", Custom Extension, Text και επικόλληση μέσα στο text box. Δεν ξεχνάμε να κάνουμε check στο Run without sandbox, πριν πατήσουμε το κουμπί Load.
